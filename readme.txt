Project - Address Finder App
Author - Anant Gwari (https://github.com/AnantGwari)
Language - JAVA, XML

The following is an android studio program. The program used android framework provided by google and android studio. The app purposes is to retrieve information from the main activity, parse it into an Intent and pass it down to google maps and/or browser. The app redirects user to the default maps application and show the results.
